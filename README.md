# eriv
a place to upload and share audio clips; [frontend](https://eriv.netlify.app/) hosted on netlify and backend on heroku

## client

- react 
- redux w/ redux-thunk
- axios
- formik
- styled compoenents
- jwt auth

## server

- node.js w/ express.js
- mongodb
- aws s3
- multer
- bcrypt
- jwt auth

## client-mobile (coming soon)

- react native
